Detailed Description

This file contains the motor control code which runs on the MCU.

Sets up 2 motors and runs motor control tasks.

This program contains 3 tasks, two tasks for motor control, and one for sending data back to the computer. The tasks are run based on priority, with the motor control tasks having higher priority.
